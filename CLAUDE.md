# Tatiana Venturacci — Landing page (Personal Trainer)

## O que é este projeto
Landing page da personal trainer **Tatiana Venturacci**.
O site está **no ar** em **https://tatianapersonal.com.br**, publicado via **GitHub Pages** a partir do repositório `henriqueizzo/tatianapersonal` (branch `main`).

## Estrutura
- `index.html` — o site inteiro em um único arquivo (HTML + CSS + JS juntos)
- `CNAME` — domínio customizado do GitHub Pages (não apagar!)

## Seções do site (ids no HTML)
`#hero` (capa) · `#modalidades` · `#planos` · `#sobre` · `#depoimentos` · `#localizacao`

## Detalhes importantes
- Todos os botões de contato apontam para o **WhatsApp** (links `wa.me`); há link do Instagram no rodapé
- **Google Tag Manager** instalado: container `GTM-M3P8K7GP` (não remover ao editar o `<head>` e o `<body>`)
- Menu mobile controlado por JS (`#menuBtn`, `#mobileMenu`, `#menuClose`)

## Como publicar alterações
Qualquer `git push` na branch `main` publica o site automaticamente (GitHub Pages). Pode levar 1–2 minutos para aparecer no ar.

O dono do projeto é **aprendiz de programação**. Sempre responda em **português do Brasil**, com explicações simples, visuais e diretas, evitando jargão técnico desnecessário.
